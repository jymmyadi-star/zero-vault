# EU Medical Device Regulation (MDR) — Zero Vault

**Regulation:** EU MDR 2017/745
**Date:** 2026-05-25

---

## Classification

**Zero Vault is NOT a medical device under EU MDR 2017/745.**

Zero Vault is a password manager, seed phrase vault, and secure notes application. It does not store, process, or analyze any health or medical data. It has no clinical, diagnostic, monitoring, or therapeutic functionality.

**No MDR assessment is required.**
